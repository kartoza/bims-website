# Carousel Headers

Here, users can edit and customise the carousel of the BIMS platform. 

![BIMS Theme-carousel headers](<img/Screenshot 2025-09-22 102411.png>)

The carousel is displayed at the top of the landing page as shown below in the red rectangle and includes the image and text. 

![FBIS Africa carousel](<img/Screenshot 2025-10-17 101608.png>)

### Steps:
- In the BIMS theme admin table, select Carousel Headers. If you need to create a new carousel, click '+ Add carousel header'
- Browse the image file that will be displayed behind your text in the carousel and populate the Title and Paragraph fields as desired.

![Add carousel-1](<img/Screenshot 2025-10-17 102208.png>)

The screenshot below shows the details used to create the carousel for FBIS Africa shown at the start of the guide. 

![Add carousel-2](<img/Screenshot 2025-10-17 102957.png>)

The remaining fields allow the user to customise the banner positioning, text and formatting. 

- Banner font drop shadows can be added here: Home > BIMS Theme > Carousel Headers > **Overlay**

![Carousel-overlay](<img/Screenshot 2025-10-17 104708.png>)

- Banner image display size can be adjusted here: Home > BIMS Theme > Carousel Headers > **Banner Display**

![Carousel-banner display](<img/Screenshot 2025-10-17 105046.png>)

- Banner title text font and position can be customised here: Home > BIMS Theme > Carousel Headers > **Title Typography and Position**

![Carousel-title topography](<img/Screenshot 2025-10-17 105248.png>)

- Banner description text font and position can be customised here: Home > BIMS Theme > Carousel Headers > **Description Typography and Position**

![Carousel-description typography](<img/Screenshot 2025-10-17 105456.png>)
