# Chart Colors

Here, users can customise the colors that will be used for the charts in the Multi-site overview dashboard. 

![Chart colors 1](<img/chart-colors-1.png>)

Guides on how to change the global conservation status chart colors are outlined in the [Manage Admin Tables > IUCN Status](https://kartoza.github.io/bims-website/administrator/biodiversity-data/manage-admin-tables/#bims-iucn-status) guide. 

To change the remaining colors, go to Admin > BIMS Theme > Chart Colors

![Chart colors 2](<img/chart-colors-2.png>)

Click 'Add chart color' and fill in the desired HEX code.

![Chart colors 3](<img/chart-colors-3.png>)

Click 'Save'. 

The order that you add the colors will be the order that it is selected in the charts. You cannot choose which colors are used for different chart categories or elements. 

The intention of the chart color customisation is to align the charts according to the platform's color scheme.
