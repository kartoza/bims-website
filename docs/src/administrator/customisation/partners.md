# Partners

Partner logos and links can be added to the system under BIMS Theme > Partners. 

![BIMS Theme-Partners](<img/Screenshot 2025-10-17 110320.png>)

To add new partner information, Click '+ Add partner' 

Then insert the name of the partner. This will be displayed when a user hovers over the logo/icon on the landing page. Browse your PC for the logo (.png) and add a link to the partner website. 

![BIMS add a partner](<img/Screenshot 2025-10-17 110858.png>)

Click 'Save'. Repeat as necessary for all partners. 

The logos will now be available on the platform and are ready to be added to the [BIMS custom theme](./custom-themes.md).
