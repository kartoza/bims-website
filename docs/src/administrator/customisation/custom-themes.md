# Custom Themes

The BIMS Custom Theme can be used to apply and design themes for landing pages on the platform

![BIMS-custom theme](<img/Screenshot 2025-10-17 111917.png>)

If you have not created a theme yet, click “+ Add custom theme”

### General

Populate the fields as outlined and browse and select and image (.png) to be uploaded as the site logo. Once uploaded, click 'Is enabled'.

![Theme-general](<img/Screenshot 2025-09-22 083703.png>)

### Navbar

The Navbar section can be used to customise the menu bar at the top of the landing page as shown below. 

![Theme-navbar-1](<img/Screenshot 2025-09-22 083859.png>)

Features such as background colour, text colour and size, and logos can be edited here. Additionally, if users require a divider to the right of the logo/brand, it can be edited, removed and enabled in this section. 

![Theme-navbar-2](<img/Screenshot 2025-09-22 084016.png>)

### Landing page

Select the landing page section under 'Available landing page sections', which will then become highlighted (see [Landing page customisations](./landing-page-customisation.md) on how to add sections). Then click the right arrow to choose the sections that will be visible on the landing page in your custom theme. You can remove sections by selecting them under 'Chosen landing page sections' and click the left arrow. 

![Theme-landing page-1](<img/Screenshot 2025-09-22 084452.png>)

Users can also select the carousel that will be presented on the landing page (see [Carousel headers](./carousel-headers.md) for how to add and edit a header)

![Theme-landing page-2](<img/Screenshot 2025-09-22 085119.png>)

### Summary Dashboard

The summary dashboard is displayed after the carousel on the landing page and includes details on the number of records and sites per biodiversity module (eg, fish, amphibians etc). Each biodiversity module icon is surrounded by a circular ring that visually depicts the internation IUCN conservation statuses of the records. 

![Theme-summary dashboards-1](<img/Screenshot 2025-09-22 085625.png>)

Here, users can also adjust the fonts, background colour and palette on the conservation status ring around the module icons. 

![Theme-summary dashboards-2](<img/Screenshot 2025-09-22 085834.png>)

### Partners

The Partners & Funders section is located at the bottom of the landing page (see [Partners](./partners.md) on how to add partners and logos to the platform). 

![Theme-partners-1](<img/Screenshot 2025-09-22 095613.png>)

The font for the partner section can be adjusted here. Users can also choose which partner and funder logos are displayed in this section, following the same workflow described above for the [Landing page](#landing-page)

![Theme-partners-2](<img/Screenshot 2025-09-22 090646.png>)

![Theme-partners-3](<img/Screenshot 2025-09-22 090722.png>)

### Colors & Buttons

This section can be used to customise the colour theme for the BIMS website.

![Theme-colors-1](<img/Screenshot 2025-09-22 090855.png>)

This can be achieved by clicking on the coloured box and selecting a colour, or by editing the HEX code.

![Theme-colors-2](<img/Screenshot 2025-09-22 091131.png>)
